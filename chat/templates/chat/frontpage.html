Welcome to Chat Lounge

A Chat Room Web Application developed in Python using the Django framework.

The application uses Channel Layers, in the back-end, and JavaScript WebSockets, in the front-end, to connect different users together, allow them to write text messages, and send them for everyone else in the room to see.

It also uses a Redis server, built and ran using Docker, as cache for the Channel Layers. This is the only officially supported cache for production use, as explained in the official documentation.

The web server of choice is Daphne, a server suitable for ASGI applications.

For more information: GitHub repository

Current features:

  • Sending and receiving text messages
  • Signing up, logging in, and logging out
  • Multiple chat rooms
  • Displaying the number of online users
  • Responsive and easy-to-use UI
